SimpleSlide

原生Javascript写的简易轮播程序，代码3KB，压缩后仅有2KB。

使用方法：

#####1.HTML中定义

	<ul class="slide">
		<li>
			<img src="" />
		</li>
		....
	</ul>

#####2.在`</body>`之前加入：

	<link rel="stylesheet" type="text/css" href="SimpleSlide.css" />
	<script src="SimpleSlide.min.js"></script>

友情提示：

js文件的最后可以设置轮播元素的Class以及轮播的速度。
